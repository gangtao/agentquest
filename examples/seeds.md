# Example World Seeds

## 1. Dark Fantasy: The Rotting Monarchy
You can run this with:
`agentquest generate --seed "A dark fantasy world where a magical rot is slowly consuming the land. The ruling monarchy has isolated themselves in a floating citadel, leaving the peasants to fend for themselves against twisted, mutated creatures."`

## 2. Sci-Fi: Neon Drift
You can run this with:
`agentquest generate --seed "A cyberpunk megacity built on the ruins of a massive ancient generation ship that crash-landed on an icy planet. Factions fight over rare 'source cores' left over from the ship's hyperdrive."`

## 3. High Fantasy: The Shattered Isles
You can run this with:
`agentquest generate --seed "A vibrant archipelago where each island possesses a unique elemental biome (fire, ice, jungle, crystal). Travel between islands is dangerous due to massive sea leviathans that demand tribute."`
